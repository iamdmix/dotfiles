# 🐧 dmix / dotfiles

My personal configuration files for macOS. Optimized for productivity, tiling window management, and a clean terminal experience.

## 🛠️ Components

| Component | Description |
| :--- | :--- |
| **[AeroSpace](https://github.com/nikitabobko/AeroSpace)** | i3-like tiling window manager for macOS. |
| **[Karabiner-Elements](https://karabiner-elements.pqrs.org/)** | Low-level keyboard customizer (Complex Modifications). |
| **Zsh** | My primary shell with custom aliases and prompt. |

## 📂 Structure

```text
.
├── aerospace
│   └── aerospace.toml    # Tiling window manager config
├── karabiner
│   └── karabiner.json    # Keyboard remapping rules
└── zsh
    └── .zshrc            # Shell environment & aliases
